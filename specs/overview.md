# Todo App - Hackathon Phase 2 & 3

## Purpose
AI-powered todo application with chatbot interface for natural language task management.

## Current Phase
Phase II & III: Full-Stack Web Application + AI Chatbot

## Tech Stack
- **Frontend**: Next.js 15, TypeScript, Tailwind CSS (Purple Theme)
- **Backend**: FastAPI, SQLModel, Neon PostgreSQL
- **Auth**: Better Auth with JWT
- **AI**: OpenAI Agents SDK, Official MCP SDK, OpenAI ChatKit

## Features
### Phase 2 - Basic Level
- [x] Task CRUD operations
- [x] User authentication (Better Auth)
- [x] Purple theme UI
- [x] Responsive design

### Phase 3 - AI Chatbot
- [x] Natural language task management
- [x] MCP server with task tools
- [x] Stateless chat endpoint
- [x] Conversation persistence

## Design Theme
- Primary Color: Purple (#9333EA)
- Secondary: Purple shades
- Modern, clean UI with smooth animations
