# UI Components - Purple Theme

## Design System

### Colors
```css
Primary Purple: #9333EA
Light Purple: #A855F7
Dark Purple: #7E22CE
Purple Background: #F3E8FF
Purple Text: #581C87
```

### Components

## 1. TaskCard
Purple-themed card for displaying tasks.

**Features:**
- Purple border on hover
- Checkbox with purple check
- Strike-through for completed tasks
- Edit/Delete buttons with purple icons
- Smooth hover animations

## 2. TaskForm
Form for creating/editing tasks.

**Features:**
- Purple input borders on focus
- Purple submit button
- Validation with purple error messages
- Smooth transitions

## 3. ChatInterface (ChatKit)
AI chatbot interface with purple theme.

**Features:**
- User messages: Purple background (#9333EA)
- AI messages: Light purple background (#F3E8FF)
- Purple send button
- Smooth message animations
- Auto-scroll to latest

## 4. Navbar
Top navigation with purple branding.

**Features:**
- Purple logo/title
- User profile with purple dropdown
- Logout button
- Responsive mobile menu

## 5. Button
Reusable purple button component.

**Variants:**
- Primary: Purple background
- Secondary: Purple outline
- Danger: Red with purple tint

## Animations
- Fade in on load
- Smooth hover transitions
- Slide in for new items
- Bounce for success feedback
